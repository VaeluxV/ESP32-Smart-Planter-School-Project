# ESP32 Smart Planter
## Info
### Hardware:
* ESP32 Devkit (30-pins)
* Soil moisture sensor
* PN532 rfid reader
* Passive buzzer
* LDR
* Ds18b20 Soil temperature sensor
* Water Level Sensor
* Bme280
* 5V 4-channel relay module
* Wire
### Optional hardware:
* Custom PCB
